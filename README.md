Blackjack (Python)

A simple terminal-based Blackjack game built in Python.

This project was created as practice for working with functions, loops, lists, and basic game logic while learning Python.

How the Game Works
	•	The player and dealer are both dealt two cards.
	•	The player can choose to hit (draw another card) or stand (end their turn).
	•	The dealer automatically draws cards until reaching 17 or higher.
	•	If a player goes over 21, they bust and lose the round.
	•	The winner is determined by comparing the final scores.

Features
	•	Random card drawing using the random module
	•	Player decision system (hit or stand)
	•	Dealer AI that follows Blackjack rules
	•	Score comparison to determine the winner
	•	Option to play multiple rounds

Technologies Used

	•	Random module
	•	Lists
	•	Functions
	•	Loops
	•	Conditional statements
